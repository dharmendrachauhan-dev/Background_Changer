# React + Vite

# Tools Used
This Project is made react + Vite and Tailwind

# How To Install and Run the Project on your Device

# Step-1
1) You can download whole folder through download just above the repo.

# Step-2
2) Repo Open in vs code and open terminal in vs code and run command (npm install) that will install what necessary package.

# Step-3
3) Again run Another command for local run (npm run dev)

# Step-4

4) Now Web is live local on your laptop
